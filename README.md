# 实验室管理系统

软件设计与构造，~~小组~~作业。

Vue3 前端 + Express 后端。

## 运行

前端用的 Vue3 + Elment Plus，在 ./vue 文件夹，后端用的是 node.js 的 express， 在 ./backend 文件夹。

### 前端

```bash
npm install
npm run dev
```

默认在 5173 端口。

### 后端

```bash
npm install
npm start
```

默认在 3000 端口。

前端的 ./src/config.js 是我定义请求 url 前缀的地方，如果你需要改端口或者用服务器运行后端，可以修改。
